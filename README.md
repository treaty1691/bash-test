# Bash Project

A small Bash shell scripting workspace with a standard layout for VS Code.

## Structure

- `scripts/` — bash scripts
- `tests/` — simple shell-based tests
- `.vscode/` — VS Code settings and tasks

## Quick Start

1. Run the main script:
   ```bash
   bash scripts/main.sh
   ```
2. Run the error-handling demo:
   ```bash
   bash scripts/bash-error-techniques.sh --demo
   ```
3. Run tests:
   ```bash
   bash tests/test_main.sh
   bash tests/test_bash_error_techniques.sh
   ```

## Scripts

- `scripts/main.sh` — entry point
- `scripts/utils.sh` — helper functions
- `scripts/bash-error-techniques.sh` — demonstrates Bash error-handling techniques, traps, timeout handling, and safe scripting patterns
